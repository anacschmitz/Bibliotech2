- Admin: cadastra os livros na biblioteca;
- Cadastro dos alunos: Próprio aluno faz o seu cadastro;
- Aluno pode pegar um livro por vez apenas;
- Para Solicitar o empréstimo e devulução o aluno deve estar logado;
